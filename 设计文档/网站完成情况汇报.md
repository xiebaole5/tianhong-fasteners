## 四、当前可用的使用方式

- **本地预览**：在项目根目录执行 `python -m http.server 8000` 或 `php -S localhost:8000`，浏览器访问 `http://localhost:8000`
- **新增产品**：编辑 `data/products.json`，在 `products` 中追加一条，并将对应图片放到 `images/products/`；若产品页尚未改为读 JSON，需同步改 `products.html` 中的静态卡片
- **上线**：推送到 GitHub 并启用 Pages，或按 `DEPLOYMENT.md` / `MANUAL_DEPLOYMENT.md` 做手动/国内部署
